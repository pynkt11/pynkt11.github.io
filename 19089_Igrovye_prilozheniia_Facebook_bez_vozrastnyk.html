Facebook представляет игровые приложения без ограничений по возрасту

Компания, имеющая в Британии тысячи игорных терминалов с фиксированной процентной ставкой (FOBT), предлагает приложения для азартных игр на Facebook без проверки возраста, что вызывает утверждения о том, что дети подвергаются риску игорной зависимости.

Ранее в этом месяце отраслевой регулятор заявил, что более 60000 детей были либо азартными игроками, либо рискуют ими стать. Эксперты предупреждают, что игры, имитирующие реальную азартную игру, являются «фактором риска номер один» для развития проблемы позже.

Scientific Games, американская компания, которая предоставила FOBT для Ladbrokes и игры в казино для нескольких сайтов азартных игр, предлагает множество «социальных игр», доступных в виде приложений на Facebook. В одном из его приложений представлены детские мультяшные персонажи The Flintstones, героиня сказки Рапунцель, а также так называемые OMG! Котята.

Социальные игры оказались среди возникающих рисков, выделенных на этой неделе в отчете Комиссии по азартным играм. Игорный регулятор установил, что дети, которые играли в социальные игры, с большей вероятностью ставят деньги на азартные игры для взрослых.
