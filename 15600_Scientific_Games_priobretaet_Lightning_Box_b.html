Scientific Games покупает Lightning Box

Гигант производства программного обеспечения для азартных игр Scientific Games приобрела сиднейского независимогопроизводителя игр для онлайн-казино Lightning Box, что еще больше поможет ему реализовать свое широкое видение и стать ведущим кроссплатформенным глобальным поставщиком контента для индустрии азартных онлайн игр.

Компания также отметила, что эта сделка знаменует собой важный шаг вперед в реализации стратегического плана по утверждению себя в качестве провайдера азартных игр с устойчивым ростом с опорой на контент.

🔥 Получить бонусы

Приобретение специалиста по созданию онлайн-слотов также должно помочь Scientific Games продолжать использовать самые большие возможности роста, с которыми она сталкивается, с упором на рынки онлайн игр.Lightning Box присоединится к множеству собственных студий, которые производят контент исключительно для Scientific Games и ее партнеров-операторов.

Партнерство между двумя компаниями будет способствовать продолжающейся интеграции Lightning Box в деятельность ее новой материнской компании и позволит ей и дальше сосредоточиться на создании и доставке нового захватывающего контента, а также извлекать выгоду из обширного диапазона ресурсов Scientific Games.

О LIGHTNING BOX

Сиднейская студия была основана еще в 2004 году ветеранами разработки игр для казино Питером Козли (Peter Causley) и Дэвидом Литтлом (David Little). Lightning Box - ведущий разработчикконтента как для онлайн-индустрии, так и для наземных казино.

Разработчик имеет давние партнерские отношения по распространению контента с Scientific Games, и его продукты с самого начала были частью платформы OpenGaming последней компании в США.

Lightning Box и Scientific Games недавно отметили еще один важный этап - запуск первого контента в Западной Вирджинии через систему OpenGaming.
